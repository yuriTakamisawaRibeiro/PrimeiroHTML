O que é HTML?

HTML (Linguagem de Marcação de HiperTexto ) é o bloco de construção mais básico da web. Define o significado e a estrutura do conteúdo da web. Outras tecnologias além do HTML geralmente são usadas para descrever a aparência/apresentação (CSS) ou a funcionalidade/comportamento (JavaScript) de uma página da web.

html: Início da página html, em que engloba todo o código dela.

head: Todo codigo que será executado antes dos elementos de marcação da página.

body: Corpo do texto em que será colocado o texto, atributos, imagens, links e etc.

Tags:

(voltar)

Tags são utilizadas para informar ao navegador que tipo de estrutura é essa que está sendo construída, podendo ser títulos, parágrafos, imagens, links, entre outros.

Tags usadas em títulos:

As páginas HTML podem usar as tags para fazerem títulos e sub-títulos, por meio do "h1" até o "h6", em que respectivamente, tem maior e menor poder, não é contado o tamanho da letra, e sim a importância dela.

"h1" tem a maior hierarquia, é o título principal(recomendável ter 1 por página)

"h2 tem a segunda maior hierarquia, sendo sub-título(pode ter vários)"

"h3", "h4", "h5" e "h6", vão diminuindo sua hierarquia, não é necessário ter uma menor hierarquia do que o h6(pode ter quantos quiser)

Tags usadas em parágrafos:

"p" faz com que possíbilite o uso dos parágrafos, sempre quando for escrever um texto, use ele.

"i" faz com que seu texto fique em itálico.

"strong" faz com que seu texto fique em negrito.

"small" faz com que seu texto fique pequeno.

"u" faz com que seu texto fique sublinhado.

"mark" faz com que seu texto fique pintado de marca texto.

"sup" faz com que seu texto fique elevado.

"sub" faz com que seu texto fique rebaixado.

"blockquote" faz com que você cite uma outra fonte, exemplo,

"O homem que move montanhas, começa levantando pequenas pedras"

"del" faz com que um texto seja rasurado e outro entra em seu lugar, exemplo, gosto da cor Azulverde.

"abbr" abrevia palavras com significados grandes, ou siglas, exemplo, JS

"hr" faz com que apareça uma linha atráves da página, que dívide partes dela, como demonstrado abaixo.

Listas:

(voltar)

Listas são importissimas numa página HTML, pois organizam elementos que podem estar ordenados ou não, ambas necessitam da tag "li" dentro delas para adicionar elementos á lista.

Listas Ordenadas:

"ol" são listas que irão ter uma ordem imposta pelo HTML ou programador, começando por 1 em diante.

Exemplo de uma ol:

Lista de prêmios:

  1. Medalha de Ouro
  2. Medalha de Prata
  3. Medalha de Bronze
  4. Certificado de Participação

Pode ter uma lista dentro de outras.

  1. Turma A
    1. Amanda
    2. Yuri
    3. Petra

Listas Não-Ordenadas:

"ul" são listas que não tem ordem específica, em que o padrão de indicação de cada elemento é uma bolinha.(também pode haver uma dentro de outras listas)

Exemplo de uma ul:

Lista de compras:

  • Tomate
  • Aveia
  • Canela
  • Alecrim

Links:

(voltar)

Links são usados para levar o usúario até outros endereços.

"a" vem de anchor(âncora), que é a tag usada para links, que vem acompanhada do atributo, "href", sendo a url que você escolherá para ser o link, junto com um texto para ser clicado.

youtube

Esse link irá abrir na mesma página da principal.
